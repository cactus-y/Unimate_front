### 전국의 대학생들이 하나되는 Unimate

왜 같은 학교 학생들끼리만 교류하는가?

전국의 대학생들과 함꼐하자.

### 철저한 익명성

오직 학교 정보만을 입력 받으므로, 익명성이 유지된다.

또한 대학교 이메일로 인증을 하기에 대학생만이 사용 가능하다.
